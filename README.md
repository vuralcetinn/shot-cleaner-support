# Shot Cleaner GitHub Pages

This folder contains the public pages needed for App Store Connect.

## Files

- `support.html`
- `privacy.html`

## Suggested GitHub Repository

Create a public GitHub repository named:

```text
shot-cleaner-support
```

Upload these files to the repository root.

## Enable GitHub Pages

In the GitHub repository:

```text
Settings > Pages > Build and deployment
```

Use:

```text
Source: Deploy from a branch
Branch: main
Folder: /root
```

## App Store Connect URLs

Replace `{github-username}` with your GitHub username:

```text
Support URL:
https://{github-username}.github.io/shot-cleaner-support/support.html

Privacy Policy URL:
https://{github-username}.github.io/shot-cleaner-support/privacy.html
```
